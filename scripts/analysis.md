# Tushare 财报披露覆盖分析

## Key Insights

### 1. 披露时间规律 (D)
- Q1/Q3 季报: 99%+ 集中在 +1 月（4月/10月底前）
- Q2 半年报: 主要在 +2 月（8月），少量 +1 月（7月）
- Q4 年报: 分散在 +1~+4 月，峰值在 +3/+4（3月底、4月底）

### 2. 预告提前特性 (F)
- 预告可在报告期结束前发布（<0 列），尤其 Q2/Q4 的预告大量提前
- 年报预告最早在上年10月就开始（提前3个月）
- 2022年起 Q1/Q3 预告数量骤降（1500+ → 300-600），疑似监管政策调整取消强制预告

### 3. 季度信息密度差异
- Q4 年报信息最丰富: D≈5700, F≈2800, E≈1400
- Q2 半年报次之: D≈5600, F≈1800, E≈100-200
- Q1/Q3 季报最少: D≈5600, F≈200-400, E≈10-30

### 4. 覆盖率
- Disclosure: 近乎全覆盖 (5600+/5700 只)
- Forecast: Q4 覆盖约50%, Q2 约30%, Q1/Q3 仅5-10%
- Express: 仅 Q4 有意义覆盖 (~25%), 其他季度 <1%
